## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Asad3533/laravel-portfolio.git
```

Go to the project directory

```bash
cd laravel-portfolio
```

Install dependencies

```bash
composer install
```

Copy environment file

```bash
cp .env.example .env
```

Generate application key

```bash
php artisan key:generate
```

Run migrations

```bash
php artisan migrate
```

Start the development server

```bash
php artisan serve
```

Open

```
http://127.0.0.1:8000
```
